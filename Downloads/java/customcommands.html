CustomCommands

Description

With this plugin you can add custom commands to minecraft. Currently it only supports social commands like "/socials discord". You can change the messages in the config.yml.

Commands:
/customcommands Displays information about the plugin
/socials Displays the possible options
/socials Discord Displays the message set in the config.yml
/socials Instagram Displays the message set in the config.yml
/socials Twitter Displays the message set in the config.yml
/socials Website Displays the message set in the config.yml
